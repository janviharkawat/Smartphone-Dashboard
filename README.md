# 📱 Smartphone Market Analysis Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Power BI dashboard** that analyzes the smartphone market using real-world data.
It focuses on **price, performance, value-for-money, and hardware features** to uncover meaningful insights for **consumers and analysts**.

---

## 🎯 Objectives

* Analyze **price vs performance relationship**
* Identify **top value-for-money smartphones**
* Compare **brand positioning in the market**
* Understand how **hardware features impact performance**
* Enable **data-driven buying decisions**

## 🧠 Key Insights

* Price does not always correlate with performance
* Mid-range phones often provide the best value
* RAM and processor are key performance drivers
* Camera megapixels are not the sole indicator of quality
* Battery capacity has standardized across the market

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers (Brand, Price Category) to filter data
3. Hover over visuals for detailed insights
4. Navigate across pages for different analyses
